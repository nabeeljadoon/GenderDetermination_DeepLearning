Project Overview

This project focuses on gender classification using deep learning, leveraging the VGG-19 model applied to eye images.

Dataset

Eye images are used for classification, extracted from well-known datasets.

Our Methodology

We utilize CNN-based feature extraction and classification using the VGG-19 model.

Results

Our model achieved 97% accuracy on gender classification.
